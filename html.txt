HTML ver1 - использовался только для структурной разметки данных.
HTML ver2- Добавили формы с наборами пользовательских элементов управления, которые использовались для ввода пользователем параметров.
           поиск по ключевым словам, ввести дату рождения или выбрать один из нескольких вариантов в опроснике
HTML ver3- которая дополнила язык разметками таблицами, разметки математических формул, фреймами, 
           изображениями, обтекание изображений текстом и некоторыми другими важными тегами. 
    ver3.2 (и тогда же был создан CSS (Cascading Style Sheets), для отделения параметров графических стилей отображений элементов в браузере)
HTML ver4- с появлением CSS, избавили HTML от тех ненужных элементов, которые  стали устаревшими и компрометировали идею отделения разметки структуры от параметризации представления.
           Но Основное достижение HTML ver4 – это появление объектной модели страницы (Document Object Model, DOM), 
           элементами которой теперь можно было манипулировать посредствам скриптовых языков программирования.
           Самым популярным таким языком программирования является JavaScript. Dynamic HTML или просто DHTML, который ознаменовал прорыв в веб-дизайне. 
HTML ver5- Улучшилась поддержка мультимедиа-технологий,Исключена еще часть устаревших тегов,
           Стало больше внимания уделяться поддержке скриптов, например javascript
